    Space Trader for Windows

    Space Trader for Windows is based closely on Space Trader for Palm, by Pieter Spronk. Much of the information on this site was taken from Pieter's Space Trader site, with permission. Visit the Download page to download the game. The game requires the .NET framework 3.5, which is available from Microsoft.
